4. 你的动态将出现在社区中

- 点击社区动态下的 **❤️** 按钮

## 📁 项目文件说明

### 核心文件
- `src/App.tsx` - 主应用入口，包含路由和Redux配置
- `src/pages/` - 5个主要页面（首页、音乐库、社区、演唱会、个人中心）
- `src/components/` - 可复用组件（导航栏、页脚、播放器、登录框）
- `src/store/` - Redux状态管理（音乐、用户状态）

### 配置文件
- `package.json` - 项目依赖和脚本
- `tailwind.config.js` - 紫色主题配置
- `tsconfig.json` - TypeScript配置
- `vite.config.ts` - Vite打包配置

## 🎨 紫色主题色

项目使用统一的紫色主题：
- **主色紫**: `#a855f7` (Purple-500)
- **深色背景**: `#030712` (Slate-950)
- **卡片背景**: `#1e293b/50` (Slate-800/50)

所有按钮、输入框、卡片都使用统一的样式类：
- `.btn-primary` - 主要按钮（紫色渐变）
- `.btn-secondary` - 次要按钮（紫色边框）
- `.card` - 卡片组件（带边框和阴影）
- `.input-field` - 输入框（统一风格）

## 🚀 部署准备

### 构建生产版本
```bash
npm run build
```
输出文件在 `dist/` 目录

### 部署到Vercel（推荐）
```bash
npm install -g vercel
vercel
```

### 部署到其他平台
1. 运行 `npm run build`
2. 将 `dist/` 文件夹上传到托管服务
3. 配置服务器重写所有路由到 `index.html`

## 🤝 需要帮助？

### 常见问题

**Q: 端口被占用怎么办？**
```bash
npm run dev -- --port 3000
```

**Q: 如何修改紫色主题？**
编辑 `tailwind.config.js` 中的颜色配置

**Q: 如何添加新页面？**
1. 在 `src/pages/` 创建新文件
2. 在 `App.tsx` 添加路由
3. 在 `Header.tsx` 添加导航链接
